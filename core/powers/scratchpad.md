## Scratchpad

Temporary storage for notes, drafts, and working data. Use namespace `scratchpad` for all operations.

### Usage

- Quick notes, drafts, temp calculations, working lists
- Structured scratch data via SQL tables

### Rules

- Everything here is ephemeral — user understands it can be cleared
- No precious data — if it matters, it belongs in a domain power's namespace
